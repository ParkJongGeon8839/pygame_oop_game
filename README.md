# 🎵 DDR 리듬게임 과제 - README

⚠️ 이 프로젝트는 **Pygame**을 활용하여 제작한 DDR 스타일 리듬게임 과제로,  
상업용이 아니며 학습/과제용 목적으로만 사용됩니다.

---

## 📋 게임 소개

화면 위에서 떨어지는 화살표에 맞춰 **방향키(↑↓←→)**를 누르며 콤보와 점수를 쌓는 리듬게임입니다.  
노래마다 난이도(Easy / Normal / Hard)와 속도(1x / 1.5x / 2x)를 선택할 수 있으며,  
**체력바(HP)가 0이 되면 게임 오버**가 됩니다.

---

## 🕹️ 기본 조작

<img src="./images/Key.png">

- **방향키(V R H U M)**: 해당 화살표 입력
- **스페이스바**: 게임 시작 / 일시정지
- **ESC 또는 창 닫기**: 게임 종료

---

## 💡 게임 시스템

### 체력 & 콤보

- HP: 0~100, 0이 되면 게임 오버
- 콤보: 연속 정확 입력 시 점수 증가
- 판정 종류: Perfect / Great / Good / Miss

### 난이도 & 속도

- 난이도 선택: Easy / Normal / Hard
- 속도 설정: 1x / 1.5x / 2x
- 난이도와 속도에 따라 화살표 등장 간격과 속도 변경

### 점수 시스템

- Perfect: 높은 점수 + HP 소량 회복
- Great: 중간 점수 + HP 소량 회복
- Good: 기본 점수
- Miss: 점수 없음, HP 감소
